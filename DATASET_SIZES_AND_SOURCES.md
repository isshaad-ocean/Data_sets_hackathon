## Dataset Download Sources

### Challenge 1 — AI Agent Creation Platform for Autonomous Crypto Trading
- **Kraken OHLCV**: CryptoDataDownload (`https://cryptodatadownload.com/data/kraken/`)
- **CryptoSentiment**: Zenodo (`https://zenodo.org/records/7684409`)
- **Bitcoin Historical**: Kaggle (`mczielinski/bitcoin-historical-data`)
- **Crypto News**: Kaggle (`oliviervha/crypto-news`)
- **Fear & Greed Index**: Kaggle (`l3llff/bitcoin`)

### Challenge 2 — Network Anomaly Root-Cause Assistant
- **NSL-KDD**: Kaggle (`hassan06/nslkdd`)
- **UNSW-NB15**: Kaggle (`dhoogla/unswnb15`)
- **LogHub (HDFS/BGL)**: HuggingFace Datasets (`aioha/loghub`)

### Challenge 3 — Continuous KYC Autonomous Auditor
- **OpenSanctions**: Direct download (`https://data.opensanctions.org/datasets/latest/default/targets.simple.csv`)
- **OFAC SDN List**: US Treasury (`https://www.treasury.gov/ofac/downloads/sdn.csv`)
- **EU Consolidated Sanctions**: EU Portal (`https://webgate.ec.europa.eu/fsd/fsf`)
- **EUR-Lex**: HuggingFace Datasets (`coastalcph/lex_glue`)
- **PrivacyQA**: HuggingFace Datasets (`allenai/privacy_qa`)
- **GDPR Text**: GitHub (`https://raw.githubusercontent.com/nickmvincent/gdpr_text/master/gdpr.json`)
- **OPP-115**: GitHub (`https://github.com/citp/privacy-policy-annotated`)

### Challenge 4 — Contract & SOW Risk Analyzer
- **CUAD**: HuggingFace Datasets (`theatticusproject/cuad`)
- **ContractNLI**: HuggingFace Datasets (`stanfordnlp/contract_nli`)
- **LEDGAR (LexGLUE)**: HuggingFace Datasets (`coastalcph/lex_glue`)
